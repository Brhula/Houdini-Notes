
### From MAYA UI:

`F9`  : Select POINTS       // `CONTROL` + `F9` : Convert selection to POINTS   
`F10` :        EDGES        // `CONTROL` + `F9` :                      EDGES   
`F11` :        PRIMITIVES   // `CONTROL` + `F9` :                      PRIMITIVES   
`F12` :        UV VERTEX    // `CONTROL` + `F9` :                      VERTEX   

`CONTROL`+`E` : Poly Extrude
